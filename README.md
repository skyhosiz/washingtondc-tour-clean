# Washington D.C. Tour System

Full-stack tourism web application for Washington, D.C.

Live demo: https://washingtondc-tour-clean-1.onrender.com

## Tech Stack

- Node.js, Express 5
- MongoDB
- JWT, bcryptjs
- Brevo (email), Cloudinary (image upload)
- HTML, CSS, JavaScript
- Security: helmet, rate limiting, cors, mongo-sanitize

## Features

- User register / login / logout
- JWT auth + protected pages
- Forgot / reset password via email
- Profile edit + avatar upload
- Explore D.C. places (museums, landmarks, food)
- In-app “D.C. Assistant” guide
